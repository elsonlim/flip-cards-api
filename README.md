# flip-cards-api

1. Create a **.env** file, an example is provided below

```text
PORT=5000
MONGO_URL=mongodb://localhost:27017/flipcards
JWT_PW="please use a secure password"
COOKIE_SECRET="please use a secure password"
```

2. Run test coverage

```bash
npm install
npm run testc
```

3. Start

```bash
npm run start
```
